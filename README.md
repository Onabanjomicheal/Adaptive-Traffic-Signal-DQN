## 📌 Overview

A Deep Reinforcement Learning agent that learns optimal traffic signal timing policies using Deep Q-Networks (DQN), simulated in SUMO (Simulation of Urban MObility) and calibrated for Lagos, Nigeria traffic patterns. Reduces average waiting time and vehicle emissions at intersections.

**Tech Stack:** `Python` `SUMO` `DQN` `Deep RL` `TraCI` `Urban Mobility`

---

## 🚀 Getting Started

### Prerequisites

```bash
pip install -r requirements.txt
```

> Requires **Python 3.8+**

### Installation

```bash
git clone https://github.com/Onabanjomicheal/Adaptive-Traffic-Signal-DQN.git
cd Adaptive-Traffic-Signal-DQN
pip install -r requirements.txt
```

### Run

```bash
python main.py
```
